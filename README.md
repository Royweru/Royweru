## ⚙️ Engineering Philosophy

- **Bottom-up, isolated testing** — every component proven on its own before it's wired into a system.
- **`httpx` over `requests`** — async-first, by default.
- **SQLite for v1, PostgreSQL as the upgrade path** — don't pay for scale you don't have yet.
- **No hallucination, no assumption** — especially in agentic systems, where confident wrong answers are worse than no answer.
- **Fix agents through prompting, not manual patches** — if the model's behavior is wrong, the instruction is wrong; edit that, not the output.
